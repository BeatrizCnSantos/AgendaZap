# AgendaZap

Sistema SaaS de agendamento online com integração ao WhatsApp.

## Tecnologias

- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- Swagger
- BCrypt

## Funcionalidades

- Cadastro de usuários
- Login de usuários
- Gerenciamento de serviços
- Controle de agenda
- Agendamento online
- Integração com WhatsApp
